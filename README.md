EmoSense Chatbot-
An intelligent emotion detection chatbot that understands user feelings and responds with empathetic and human-like replies.

About the Project-
EmoSense is a chatbot designed to detect human emotions from text input and provide appropriate responses. It uses Natural Language Processing (NLP) techniques and intent classification to identify emotions like happiness, sadness, anger, fear, and neutrality.

The goal of this project is to create a supportive conversational agent that can interact with users in a more human and emotionally aware way.

- Features
 Detects multiple emotions (Happy, Sad, Angry, Fear, Neutral)
 Provides empathetic and context-aware responses
 Real-time chatbot interaction
 Dialogflow integration for intent classification
 Flask backend for webhook handling
 
-Tech Stack
Python 🐍
Flask (Backend API)
Dialogflow (NLP & Intent Detection)
Streamlit (Optional UI)
JSON (Data Handling)
